# CityAssist Frontend (PWA Skeleton)

Run locally:
1. npm install
2. npm run dev
3. Open http://localhost:3000

Features included:
- Onboarding / profile setup (PWA-first)
- Home feed with personalized alerts (mocked)
- Report an issue (photo capture/upload simulated)
- Service status subscription UI
- Local services directory
- PWA manifest and simple service-worker for offline caching
- Mock auth via app/api/auth/login (reads data/users.json)
- Mock data in /data for offline demo
